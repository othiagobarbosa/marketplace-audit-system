# marketplace-audit-system
# 🔍 Sistema de Auditoria Financeira para Marketplaces

Sistema automatizado que integra via API com **Shopee**, **Shein** e **Mercado Livre** para identificar divergências financeiras em tempo real.

## 🎯 O Problema
Sellers de marketplaces perdem dinheiro todo mês sem saber:
- Taxas cobradas incorretamente
- Comissões duplicadas
- Repasses atrasados ou com valores errados
- Divergências entre extrato da plataforma e valor recebido

Nenhuma ferramenta acessível resolvia isso de forma automatizada.

## ✅ A Solução
Sistema próprio que cruza automaticamente os dados financeiros das 3 plataformas e gera relatórios de auditoria detalhados.

## ⚙️ Como Funciona
1. **Conexão via API** — Shopee Open Platform, ML API, Shein API
2. **Coleta automática** de vendas, comissões, taxas e repasses
3. **Motor de auditoria** — cruzamento e identificação de anomalias
4. **Dashboard** com alertas e histórico de divergências
